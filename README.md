## Installation hints

### Node.js

1. Get nvm: https://github.com/nvm-sh/nvm
2. Make sure to add the nvm env vars at the end of your .bashrc
3. Install Nodejs:
   ```
   nvm install v10.19.0
   nvm use v10.19.0
   nvm alias default v10.19.0
   ```
4. Edit start.sh
5. ./start.sh or screen -dm ./start.sh
